# Environmental Impacts of Mining in the Philippines

A responsive advocacy-style website built with Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

This project is already configured for Vercel with `vercel.json`.

- Build command: `npm run build`
- Output directory: `dist`

You can import this repo directly in Vercel and deploy without extra setup.
